# Assignment-3 (Project) FAQ Summary

- Intermediate deadlines are **for reference only** — no late penalties if missed
- However, everything must be submitted before the **final project deadline**
- Reference deadlines are available on **Week-1 Introduction slides (slide 7)**

## Submission Rules

| Part | How to Submit |
|------|---------------|
| **Part-A** (Proposal) | **All members submit individually** (content can be identical) |
| **Part-B, C, D** | **One submission per team** is sufficient (all members may submit if they want) |
| **Part-E** (Final Report) | **All members submit individually** — each writes their own `Appendices: A. Individual Contribution` section; all members fill in the contribution % table and sign it |
| **Part-G** (Oral Defense) | **Mandatory for all** — covers Week 1–11 content + project. Assignment is **INCOMPLETE** without oral defense |

## Model / Framework

- Any object detection or image classification framework is allowed (e.g., YOLO)
- **Using only a pre-trained model is not allowed** — the network must be trained
- **Transfer learning is permitted**

## Contribution Disputes

- If team members cannot agree on contribution split, each may submit the report from their own perspective
- The subject coordinator will investigate and make a final decision
